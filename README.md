**Fidel Omondi | Full-Stack Developer Portfolio**

This is a Flask-powered portfolio website designed to showcase my technical skills, professional projects, and journey as a Software Engineer. The site focuses on clean aesthetics, fast performance, and a "projects-first" layout.

**🌟 Key Features**

- **Dynamic Project Showcasing**: Each project is highlighted with detailed descriptions and links to their respective repositories.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewing to ensure recruiters can access my work from any device.
- **Flask Routing**: Utilizes a clean Flask backend to handle navigation and modular template rendering.
- **Integrated Contact System**: A structured contact page designed to facilitate professional inquiries.

**🛠️ Technical Stack**

- **Backend**: Python (Flask)
- **Frontend**: HTML5, CSS3 (Custom Grid & Flexbox layouts), JavaScript
- **Deployment**: Configured for easy deployment on platforms like Render, Heroku, vercel
- **Icons & Styling**: Professional iconography from Font Awesome and clean typography pairings.

**📂 Project Structure**

`├── static/              # Portfolio CSS, profile images, and project thumbnails
├── templates/           # Modular HTML templates (base.html, index.html, projects.html)
├── run.py               # Flask application core and navigation routing
└── requirements.txt     # Necessary libraries (Flask, etc.)`
