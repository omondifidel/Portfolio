001 : Security Audit Service

In a world of data breaches, privacy is not an option—its a requirement. I built this system to prove that you can audit security without ever compromising user data

This project is a high-integrity security tool designed to verify if a user's password has been compromised in a known data leak. Built with Python, it integrates the Have I Been Pwned API using a specialized security protocol known as k-Anonymity. By utilizing SHA-1 hashing, the application ensures that the plain-text password never leaves the user's local machine. Only the first five characters of the hash are sent to the API, maintaining a zero-knowledge architecture. This project demonstrates my ability to manage API integrations, cryptographic data handling, and automated system notifications.
