002 : JPG to PNG Converter CLI

This project is a high-performance Command-Line Interface (CLI) utility designed for bulk image processing. Developed using Python and the Pillow (PIL) library, it automates the conversion of entire directories from JPG to PNG format.

Beyond simple conversion, the tool features intelligent Dynamic Path Handling to ensure cross-platform compatibility (Windows, Mac, Linux) and automatic directory creation. This project highlights my mastery of System Manipulation (sys.argv), OS Integration, and building "guard rails" through robust Error Handling to prevent system crashes during bulk operations.
